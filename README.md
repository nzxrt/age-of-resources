# Age of Resources

Игровой проект — управление вымышленными странами на карте.

## Запуск

- **Графический режим** (рекомендуется): `python main_game.py` или через `launcher.py`
- **Консольный режим**: `python terminal.py`

## Графический режим (main_game.py)

**Управление только клавиатурой, без терминала.**

1. **Старт**: Клик по нейтральному сектору → ввод имени (клавиатура) → Enter
2. **Шахта [M]**: Выбери сектор в горах/лесу (стоимость 100)
3. **Грядки [G]**: Выбери сектор на равнине/поле (стоимость 50)
4. **Собрать [E]**: Выбери сектор с постройкой
5. **Захват [C]**: Выбери соседний чужой сектор (стоимость 150)
6. **Торговля [T]**: Меню — ↑↓ выбор, Enter подтвердить, Esc выход, ←→ кол-во
7. **Война/Мир [W]**: Список стран — выбери ↑↓, Enter переключить война↔мир
8. **Союзы [A]**: Список стран — выбери ↑↓, Enter заключить/расторгнуть союз
